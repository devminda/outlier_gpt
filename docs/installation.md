# Installation

## Stable release

To install outlier_gpt, run this command in your terminal:

```sh
uv add outlier-gpt
```

Or if you prefer to use `pip`:

```sh
pip install outlier-gpt
```

## From source

The source files for outlier_gpt can be downloaded from the [Github repo](https://github.com/devminda/outlier_gpt).

You can either clone the public repository:

```sh
git clone git://github.com/devminda/outlier_gpt
```

Or download the [tarball](https://github.com/devminda/outlier_gpt/tarball/master):

```sh
curl -OJL https://github.com/devminda/outlier_gpt/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd outlier_gpt
uv pip install .
```
